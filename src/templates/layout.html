HELP

This program will parse text files created from PDF dumps obtained from CoPath Cytology-Tissue Correlation reports. For additional information contact germancz81@gmail.com.

INSTRUCTIONS

1. Copy the contents of the PDF file obtained from a CLIA Reports->Cytology-Tissue Correlation report and save the text file (*.txt).

2. Fill in the options: text file path, output project name and Copath site originating the report.

3. Run.

4. The project will be saved in the "job_output" folder which will open automatically once the job is done.

OUTPUT

1. Comma-delimited text file with all cases for a single patient on a single row ("out_single_row.csv")

2. Comma-delimited text file with one case per row ("'out_multiple_row.cs")

3. JSON formated file with PATIENT NAMES as keys and list of surgical and cytology cases as values ("out_data_elastic.json")
